# Simon Game

A classic Simon memory game built with HTML, CSS, and JavaScript.

## How to Play

1. Press the **A** key to start the game
2. Watch the sequence of colors that light up
3. Repeat the sequence by clicking the colored buttons
4. The sequence gets longer with each level
5. If you make a mistake, the game ends and you can restart

## Features

- Responsive design
- Sound effects for each color
- Progressive difficulty
- Clean and colorful interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6 Classes)
